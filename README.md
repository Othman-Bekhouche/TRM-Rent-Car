# TRM Rent Car

Plateforme web pour la gestion de location de véhicules.

## À Propos
Projet développé pour **TRM Rent Car**. 
Il comprend :
- Un site web public
- Un espace de réservation client
- Un CRM pour l'administration (gestion de flotte, réservations, etc.)

## Stack Technique
- **Frontend :** React, TypeScript, Vite, Tailwind CSS
- **Backend / Database :** Supabase (PostgreSQL, Auth, Storage)
- **Déploiement :** Docker (local) vers VPS.

Consultez le dossier `docs/` pour le détail de l'architecture, des phases de déploiement et des **[fonctionnalités implémentées](docs/features/06_all_functionalities.md)**.
